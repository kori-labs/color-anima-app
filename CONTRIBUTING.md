# Contributing

Thank you for your interest in Color Anima.

## External pull requests are not accepted

This repository is **source-available, not open-source**. Pull requests from
forks will be closed without review. We do not currently operate a Contributor
License Agreement and will not merge external code contributions.

## What you can do

- **Bug reports**: open a GitHub Issue. Please include macOS version, Xcode
  version, reproduction steps, and (if relevant) a minimal sample project.
- **Feature suggestions**: open a GitHub Issue tagged with `enhancement`. We
  read everything; we may or may not act on it.
- **Discussions about commercial use**: see the License section below.

## Repository status

This repository is in pre-release preparation. The committed Swift package is a
sanitized public shell. Maintainer-only release intake can stage an approved
binary artifact under ignored `.local-core/`, but private implementation source
is not accepted in public history.

If you have pre-release access, do not redistribute unreleased assets,
credentials, or project material.
